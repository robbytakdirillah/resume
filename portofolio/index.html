Tentang

Nama saya Robby Takdirillah, sejak tahun 20016 saya berkuliah di Universitas Muhammadiyah Sukabumi dengan jurusan Teknik Informatika. Saat ini saya telah mencapai semester 5. sebagai mahasiswa Teknik Informatika, saya sering belajar Web Developer, menggunakan PHP sebagai bahasa pemrograman utama. Biasanya saya menggunakan Laravel, WordPress, atau Codeigniter dalam proses belajar. Selain PHP, saat ini saya juga ikut berkecimpung dalam area frontend developer, menggunakan CSS dan JS secara lebih dalam.

Saat juga sebagai freelancer, saya berkerja dengan berbagai macam instansi untuk membangun berbagai macam kebutuhan yang menggunakan website sebagai platform solusinya. Sebagian besar pekerjaan yang dapat adalah dari upwork, saya bersyukur punya kesempatan untuk ikut berkontribusi pada berbagai macam projek dan startup dari berbagai negara di luar sana, beberapa diantaranya masih berlanjut sampai hari ini menjadi kontraktor tetap.
